# DAT 490 Capstone Project: BRFSS Poor Health Prediction

This repository contains the code for our DAT 490 capstone project. The project uses the 2024 Behavioral Risk Factor Surveillance System (BRFSS) dataset to predict self-reported poor health using non-clinical variables.

## Project Goal

The goal of this project is to examine whether non-clinical factors can help identify respondents at higher risk of reporting poor health. The predictors include insurance information, healthcare access, socioeconomic status, and demographic variables.

## Files

- `DAT490_Capstone_project_EDA_V2.ipynb`: Main notebook for data cleaning, feature engineering, exploratory data analysis, model training, model evaluation, and model interpretation.

## Data Source

The original BRFSS 2024 dataset is publicly available from the CDC BRFSS website. The raw data file is not included in this repository because of file size.
